css BFC

Posted on 2018-05-15

BFC是指块级格式化上下文，具有BFC特性的元素可以看作一个独立的容器，里面的元素无论怎样变化都不会影响到外面的元素。具有BFC特性的子元素不会受到外部元素影响，也不会影响外部元素

触发BFC

  • html body 元素
  • 浮动元素：float 除 none 以外的值
  • 绝对定位元素：position (absolute、fixed)
  • display 为 inline-block、table-cells、flex
  • overflow 除了 visible 以外的值 (hidden、auto、scroll)

BFC特性

  • 不同BFC元素下的margin不会发生折叠

  • BFC 清除浮动(如果不清除，子元素浮动，父元素高度塌陷，会影响到后面元素的布局和定位，这与BFC元素具有的特性想违背)

  • BFC 可以阻止元素被浮动元素覆盖
